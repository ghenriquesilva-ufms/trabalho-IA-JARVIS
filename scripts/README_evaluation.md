# Avaliacao do sistema

1. Preencha as 10 perguntas em data/evaluation_template.csv.
2. Rode o script para gerar respostas e trechos recuperados:

```bash
py scripts/run_evaluation.py
```

3. Abra data/evaluation_results.csv e preencha a coluna "classificacao" com:
- correta
- parcialmente correta
- incorreta

4. Use "observacoes" para anotar detalhes (ex: problema de recuperacao ou geracao).
